# 🔐 Memorable Password Generator

A desktop app built with Python that generates passwords you'll actually remember — no more random gibberish like `@#$%^&*`!

## 💡 How it works
Instead of random characters, the app asks you 4 simple questions:
- What is this account for?
- A word you love?
- Your lucky number?
- How do you feel today?

Then it builds a password that reads like a sentence — something your brain can naturally remember!

## 🔑 Example
| Account | Word | Number | Feeling | Password |
|---|---|---|---|---|
| Instagram | Reels | 67 | Happy | `ILoveReelsOnInstagram67` |
| Netflix | Chill | 99 | Calm | `ChillOnNetflixAlways~99` |
| Gmail | Coffee | 7 | Excited | `CoffeeMakesExcited!7` |
